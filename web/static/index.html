🎤 Acceso al micrófono

Esta app usa tu voz para controlar la base de datos de usuarios.

  • Graba audio mientras mantienes presionado el botón 🎤
  • Transcribe tu comando con OpenAI Whisper
  • Ejecuta búsquedas, creación o estadísticas
  • Responde con voz sintetizada (TTS)

El audio se procesa en el servidor y se envía a OpenAI solo para transcribir. No guardamos grabaciones en disco de forma permanente.
